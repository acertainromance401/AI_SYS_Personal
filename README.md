## 4. 기존 서비스 분석
- 현재 존재하는 경쟁 서비스
	- 빅케이스, 헌법 GPT
- 기본 서비스 한계점
	1. 빅케이스: 판결문 핵심 요약 및 판례 검색에는 용이하나, 시험 문제와의 상호작용 기능이 부족함.
	2. 헌법 GPT: 판례가 뒤집어지는 일이 거의 없는 헌법 중심으로 학습되어 있어, 최신 판례에 가중치를 두어야 하는 본 프로젝트 목적과 어긋남.
